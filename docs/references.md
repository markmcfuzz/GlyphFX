# References

Resources and projects that made GlyphFX possible.

---

## Censhine

An open source project by Sledmine that decrypted and
documented the original Halo CE shaders. The decrypted `shader_model`
was instrumental as a reference during the initial proof of concept
for this project.

- [Censhine on GitHub](https://github.com/Sledmine/censhine)

---

## Halo CE - Mod Tools (343 Industries)

343 Industries provided the original shader source code as part of the
MCC modding support initiative. The `.psh`, `.vsh` and `.fx` files
included in the official editing kit were the primary reference for
understanding the original rendering pipeline of each shader type.

- [Halo CE - Mod Tools](https://store.steampowered.com/app/1532190/Halo_CE_Mod_Tools__MCC/)
